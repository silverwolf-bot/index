Privacy Policy

USAGE OF DATA

The bot may use stored data, as defined below, for different features including but not limited to command handling and server leaderboards.
No usage of data outside of the aforementioned cases will happen and the data is not shared with any 3rd-party site or service.

STORED INFORMATION

The bot may store the following information automatically when any of its commands are used:
- ID of the Discord Server
- ID of the Discord Channel
- ID of the Discord User

No other information obtained via Discord’s API outside of the above mentioned will be stored.

UPDATING DATA

The aforementioned data may be updated when using specific commands.

Updating said data will require the input of an end user. No other actions, including commands used by other users on the end user, may update the stored information at any given time.

LINKS TO OTHER SITES

This Service may contain links to other sites. If you click on a third-party link, you will be directed to that site. Note that these external sites are not operated by us and are controlled by third parties. Therefore, we strongly advise you to review the Privacy Policy and the Terms of Service of these websites. We have no control over and assume no responsibility for the content, privacy policies, or practices of any third-party sites or services that you access through our service.

REMOVAL OF DATA AND SECURITY CONCERNS

Removal of the data can be requested under the following:

- Sending an email to silver.wolf.bot.dev@gmail.com
- Sending feedback through the /feedback command. We request that you provide your User ID and Server ID for debugging purposes.
- Contacting the bot creator directly on Discord
- Joining the Discord Support Server - https://discord.gg/ux3rH8jNw5
If you request removal of data associated with a server, you will need to provide proof of ownership of the server that you wish the data to be removed of.

If you have security concerns about your data or experience, you can contact us by using the above means.
